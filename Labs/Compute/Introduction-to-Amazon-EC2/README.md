# Introduction to Amazon EC2

## Overview
This lab provides a hands-on introduction to Amazon Elastic Compute Cloud (EC2), focusing on launching, managing, monitoring, and resizing EC2 instances in AWS.

The lab simulates deploying a web server and managing its lifecycle using AWS best practices.

---

## Objectives
- Launch an EC2 instance with termination protection enabled
- Deploy a web server using User Data scripts
- Monitor EC2 instance health and performance
- Configure security group rules to allow HTTP access
- Resize an EC2 instance and modify EBS storage
- Test and disable termination protection
- Terminate an EC2 instance safely

---

## AWS Services Used
- Amazon EC2
- Amazon EBS
- Amazon CloudWatch
- Amazon VPC
- Security Groups

---

## Lab Duration
Approximately 45 minutes
